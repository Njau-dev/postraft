# Fonts Directory

Add TrueType (.ttf) font files here:

- bold.ttf - Bold font (e.g., Roboto-Bold.ttf)
- regular.ttf - Regular font (e.g., Roboto-Regular.ttf)

You can download free fonts from:
- Google Fonts: https://fonts.google.com/
- Font Squirrel: https://www.fontsquirrel.com/

Recommended fonts:
- Roboto
- Open Sans
- Montserrat
